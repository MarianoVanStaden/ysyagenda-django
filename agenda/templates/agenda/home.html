Informacion del Proyecto

Este proyecto fue desarrollado con Django 5.2.7 y Bootstrap 5 como parte del Proyecto Final de la materia "Introduccion a la Programacion Web con Django".

Caracteristicas implementadas:

  • Vistas basadas en clases (Class-Based Views)
  • Herencia de plantillas con base.html
  • Consumo de API REST backend en Java
  • Diseno responsivo con Bootstrap 5
  • Manejo de errores y mensajes de usuario
